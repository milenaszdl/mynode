Скромная документация для скромного сервера

Обрабатываемые запросы

  • GET-запрос на корневой путь '/' возвращает приветствие
  • GET-запрос на '/stats' возвращает небольшой фрагмент в виде html таблицы, где в одной колонке user-agent, а во второй количество запросов.
  • POST-запрос на '/comments' возвращает данные из комментария в http-файле
